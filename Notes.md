## 1. Traversal Pattern

### Used In

- Maximum Element
- Minimum Element
- Sum of Array
- Count Even & Odd
- Linear Search
- Frequency Count

### Logic

Traverse the array once and perform the required operation on each element.

### Time Complexity

O(n)

### Space Complexity

O(1)
## 2. Two Pointer Pattern

### Used In

- Reverse Array
- Reverse String
- Palindrome
- Move Zeroes

### Logic

Maintain one pointer from the beginning and another from the end.
Swap or compare elements while moving both pointers toward each other.

### Time Complexity

O(n)

### Space Complexity

O(1)
## 3. HashSet Pattern

### Used In

- First Duplicate
- Unique Elements

### Logic

Store visited elements in a HashSet.
If an element already exists in the set, it is the duplicate.

### Time Complexity

O(n)

### Space Complexity

O(n)
## 4. HashMap Pattern

### Used In

- Frequency Count
- Character Frequency
- Majority Element

### Logic

Store elements as keys and their frequencies as values.

### Time Complexity

O(n)

### Space Complexity

O(n)
